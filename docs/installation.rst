============
Installation
============

Requirements
------------

* ``pytest >= 7.0.0``
* ``pytest-html >= 4.0.0``

Optional modules
----------------

* ``selenium >= 4.11.0`` if using **Selenium**.
* ``allure-pytest >= 2.13.2 if using **Allure**.


Installing pytest-webtest-extras
--------------------------------

.. code-block:: bash

  $ pip install pytest-webtest-extras
